# 🛍️ Cartza — E-commerce Clothing Website

Cartza is a modern full-stack e-commerce platform for clothing, built using the MERN stack.
It provides a seamless shopping experience for customers and a powerful admin panel for managing products, orders, and users.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/cartza.git
cd cartza
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
```

Create `.env` file in backend:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLOUDINARY_URL=your_cloudinary_url
```

Run backend server:

```
npm start
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm run dev
```
